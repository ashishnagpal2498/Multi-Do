About Us

Stepping into the world of web development and this is your first project. So what so special about this project ? This is a simple javascript project which focuses on beginner as well as intermediate level developers. It uses multiple databases such as mysql and mongoDB and Object relational model such as Sequelize, build on top of nodeJS. The main aim of this project is to understand different concepts of javaScript such as -

  • Callbacks
  • Promises
  • DOM manipulation
Also you will get to know how to
  • Create API using nodeJs
  • Handle database
